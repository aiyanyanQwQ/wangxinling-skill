## 建议的深挖方法

### 路线 A：先探第二梯队字幕

对第二梯队的 5 条先逐条跑：

```bash
yt-dlp --cookies /path/to/cookies.txt --list-subs https://www.bilibili.com/video/<BVID>/
```

### 路线 B：有 `ai-zh` 就直接落盘

```bash
yt-dlp --cookies /path/to/cookies.txt \
  --socket-timeout 20 \
  --write-subs \
  --sub-langs ai-zh \
  --sub-format srt \
  --skip-download \
  -o '/path/to/cyndiwang.skill/sources/transcripts/%(id)s.%(ext)s' \
  https://www.bilibili.com/video/<BVID>/
```

然后：

```bash
python /path/to/cyndiwang.skill/scripts/srt_to_transcript.py \
  /path/to/cyndiwang.skill/sources/transcripts/<BVID>.ai-zh.srt \
  /path/to/cyndiwang.skill/sources/transcripts/<BVID>_transcript.txt
```

### 路线 C：若第二梯队无字幕，降级处理

- 不再一股脑手工补所有无字幕视频
- 只对标题明显高价值的 1-2 条考虑人工精读
- 其余只建卡片，不投入转写
